# Emulator

You can debug your bot conversation using the built-in Emulator in the right-side panel.

<Callout type="info">
  **Note** <br/>
  To visualize the entire user experience, you should use the Web Channel directly. All the messages you send with the
  Emulator come from the same user from the bot's perspective.
</Callout>

## New Conversation

Go to the 3 dots at the top right corner of the emulator and click on New Conversation. This will erase the history and state from the old conversation and start a new conversation with the same user information.

## Start as a new user

Go to the 3 dots at the top right corner of the emulator and click on Start as new user. This will erase the history and state from the old conversation and start a new conversation with different user information.
