## Enabling Viber in Botpress Cloud Dashboard

Go to the chatbot settings in your cloud dashboard and enable Viber. We will get the Authentication token from the following steps.

1. Visit the [Viber Admin panel](https://partners.viber.com/login/) and click on **Create a Bot Account**.
2. Fill in the required details. Bot Avatar, Account Name, URI, Category, Subcategory, Language, Account Description, Email Address and Location.
3. Copy the Authentication Token and paste it in Viber's channel settings in Botpress Cloud Dashboard.
4. Add Bot Name (required) and Bot Avatar URL (optional).
5. Click **Save**.
6. Start chatting with the bot!
